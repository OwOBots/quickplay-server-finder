# how to use

1. install the required dependencies:
    ```sh
    pip install -r requirements.txt
    ```

2. run the main script:
    ```sh
    python main.py
    ```

thats it

## for local development of the web version

1. install the required dependencies:
    ```sh
    pip install -r requirements.txt
    ```

2. start the web server:
    ```sh
    flask run 
    ```
